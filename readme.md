Simple Mail
===========

Простой и удобный Gulp фреймворк для быстрой разработки адаптивных HTML писем,
вдохновленный [kilogram](https://github.com/dudeonthehorse/kilogram)

Установка
---------

Клонируйте репозиторий и запустите

~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
$ npm install
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~
